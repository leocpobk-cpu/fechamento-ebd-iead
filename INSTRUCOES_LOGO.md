# Instruções para Adicionar a Logo da Assembleia de Deus

## Passo a passo:

1. **Salve a imagem da logo** que você tem (a do círculo azul com a Assembleia de Deus de Cuiabá)

2. **Renomeie o arquivo** para `logo.png`

3. **Copie o arquivo** para a pasta: `D:\OneDrive\Desktop\IEAD\`

4. **Execute os comandos** no terminal:
```powershell
git add logo.png
git commit -m "feat: adiciona logo oficial da Assembleia de Deus"
git push origin main
```

## A logo será exibida em:
- ✅ Tela de login
- ✅ Cabeçalho do sistema
- ✅ Relatórios impressos

## Especificações técnicas:
- Formato: PNG
- Tamanho recomendado: 500x500px ou maior
- Fundo: Transparente (ou será adicionado círculo branco automaticamente)
